1.	An area is as
A)	part of an AS
B)	composed of at least two AS
C)	another term for an AS
D)	composed more than two AS
ANSWER: A

2.	Count-to-Infinity problem occurs in
A)	distance vector routing
B)	short path first
C)	link state routing
D)	hierarchical routing
ANSWER: A

3.	Distance vector routing algorithm is implemented in Internet as
A)	OSPF
B)	RIP
C)	ARP
D)	APR
ANSWER: B

4.	For centralized routing the decision is made by some designated node called as
A)	designated center
B)	control center
C)	network center
D)	network control center
ANSWER: D

5.	For purposes of routing, the Internet is divided into ?
A)	wide area networks
B)	autonomous networks
C)	local area networks
D)	autonomous system
ANSWER: D

6.	If there is only one routing sequence for each source destination pair, the scheme is known as
A)	static routing
B)	fixed alternative routing
C)	standard routing
D)	dynamic routing
ANSWER: B

7.	In distance vector routing algorithm, each router maintains a separate routing table with the following entries.
A)	preferred input line , estimated time
B)	preferred input line, estimated distance
C)	preferred output line, estimated time
